# 📱 To-Do Tasks App

Aplicación móvil desarrollada con Ionic + Angular que permite gestionar tareas y categorías utilizando almacenamiento local y Firebase Remote Config.

---

# 🚀 Tecnologías utilizadas

- Ionic 7
- Angular 17
- TypeScript
- Cordova
- Firebase
- Firebase Remote Config
- Signals API (Angular)
- LocalStorage

---

# ✨ Funcionalidades

## ✅ Gestión de tareas

- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
- Filtrar tareas por categoría

---

## ✅ Gestión de categorías

- Crear categorías
- Editar categorías
- Eliminar categorías
- Asignar categorías a tareas

---

## ✅ Feature Flags con Firebase Remote Config

La aplicación utiliza Firebase Remote Config para habilitar o deshabilitar funcionalidades dinámicamente sin necesidad de desplegar una nueva versión.
